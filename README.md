## üìò Description

This project shows an **educational implementation** of the Trapezoidal Point Location algorithm. It focuses on:

* visualizing the creation of the trapezoidal map,
* showing how segments split existing trapezoids,
* demonstrating how the query structure evolves

The implementation is intentionally simple and optimized for clarity rather than performance.
